{% load i18n %} {% comment %} Reusable "discard changes?" confirmation, driven by unsavedChanges.ts. Include it on any page that has a form marked `data-unsaved-guard`. When the user clicks a `data-unsaved-cancel` link while the form is dirty, the guard shows this modal instead of navigating; the `data-unsaved-confirm` button then leaves. Hard navigations and tab-close are caught separately by the native beforeunload prompt. {% endcomment %}
