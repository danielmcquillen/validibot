{% trans "What are signals?" %}

{% blocktrans trimmed %} Signals give meaningful names to values buried in your submission data. Instead of writing assertions against raw paths like p.project.compliance.targets.max_site_eui_kwh_m2, you define a signal called target_eui and write s.target_eui in your CEL expressions. {% endblocktrans %}

{% trans "How it works" %}

{% blocktrans trimmed %} Each signal has a name (a valid identifier like floor_area) and a data path that points to where the value lives in the submission payload. Validibot resolves the path when a validation run starts and makes the value available as s.<name> in every step's CEL assertions. {% endblocktrans %}

{% trans "Quick example" %}

{% blocktrans trimmed %} Given submission data like: {% endblocktrans %} 

{
  "building": {
    "geometry": { "floor_area_m2": 4800.0 }
  },
  "targets": { "max_eui": 120.0 }
}

{% blocktrans trimmed %} You would create two signals: {% endblocktrans %}

{% trans "Name" %} {% trans "Data path" %}
floor_area building.geometry.floor_area_m2
target_eui targets.max_eui

{% blocktrans trimmed %} Then in your assertions: s.floor_area > 0 or output.site_eui < s.target_eui. {% endblocktrans %}

{% trans "Tips" %}
  • {% blocktrans trimmed %} Use Sample Data on the right to paste real submission JSON or XML. Validibot will detect available paths you can map. {% endblocktrans %}
  • {% blocktrans trimmed %} Signal names must be valid identifiers: letters, numbers, and underscores only, starting with a letter or underscore. {% endblocktrans %}
  • {% blocktrans trimmed %} For arrays of named objects, use a filter expression in the data path: items[?@.name=='emissivity'].value. {% endblocktrans %}
  • {% blocktrans trimmed %} Set On missing to control what happens when a path can't be resolved: Error stops the run, Null injects null (guard with s.name != null in CEL). {% endblocktrans %}
  • {% blocktrans trimmed %} Signals are resolved once at the start of a validation run and stay consistent across all steps. {% endblocktrans %}
