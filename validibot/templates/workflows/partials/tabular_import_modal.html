{% load i18n %} {% comment %} "Import Table Schema" modal — a two-screen wizard launched from the page header. Screen 1 (input): paste a Frictionless Table Schema descriptor or upload one. The "Import schema" submit posts to the import endpoint; on success the endpoint returns the review body + an HX-Trigger that flips the modal to screen 2, and on failure it HX-Retargets the error onto this screen's error slot. Screen 2 (review): the compatibility report + column-count summary, with Apply (replace the columns and close), Back (return to input), and Cancel. The endpoint-rendered body carries the hidden ``pending_schema`` that Apply submits. Screen switching, Back, and Apply-close are wired in tabular_step_settings.html. {% endcomment %}
