{% load i18n core_tags static %} {% comment %} Outputs card for validation run results. Displays simulation outputs and findings (errors/warnings/info) from each validation step. Split into two sections: 1. **Workflow Outputs** — aggregated signal metrics across all steps (only shown when at least one step produced signal data). 2. **Outputs by Step** — per-step findings table grouped by severity, plus per-step signals and template parameters/warnings. Context variables (passed via ``{% include ... with %}``): step_runs — list of ValidationStepRun objects step_signals — dict mapping step_run.pk → list of DisplaySignal has_signals — bool, True when step_signals is non-empty all_signals — flat list of all DisplaySignal objects across steps step_params — dict mapping step_run.pk → list of param dicts step_template_warnings — dict mapping step_run.pk → list of warning strings bare — when truthy, render only the inner content (no card / header / body chrome). Used by the stacked accordion layout where the accordion supplies the chrome. Defaults to falsy, preserving the standalone card for the classic layout and the launch-page status card. {% endcomment %} {% if not bare %}
